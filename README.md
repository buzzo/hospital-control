# hospital-control
A full working backend and frontend CRUD example.

UI Technologies:
- angularJS
- browserify
- gulp

BE technologies:
- java 8
- spring boot
- spring web

Infrastructure:
- gradle
- docker
- docker-compose

### Quick Start
You will need Java 8, Node.js, docker and docker-compose installed in your machine.


```
cd hospital-control
./gradlew buildDocker
docker-compose up
```

Then just check http://localhost:8080
